### Zadanie

Zaimplementuj metodę `isEquationCorrect` 
w klasie `Mathematician` tak, aby sprawdzała czy `equation` 
jest matematycznie poprawne.

Dozwolone równania `a+b=c` i `a-b=c`

`a`, `b` i `c` to liczby jednocyfrowe

#### Przykłady

`"2+2=4"` -> `true`

`"9-3=6` -> `true`

`"1+1=1"` -> `false`
